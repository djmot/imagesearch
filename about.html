Image Search Abstraction Layer

Search for images, paginate results, and view recent search queries.

Search for images:

https://imagesearch-djmot.herokuapp.com/imagesearch/<searchtext>

Add the paramater "offset=n" to paginate results:

https://imagesearch-djmot.herokuapp.com/imagesearch/<searchtext>?offset=3

View recent search queries:

https://imagesearch-djmot.herokuapp.com/imagesearch/
